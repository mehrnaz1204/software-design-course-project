## 🧩 موجودیت‌های اصلی سیستم (External Entities)

| نقش | توضیح |
|-----|--------|
| راننده | درخواست‌دهنده ورود و خروج |
| دوربین پلاک‌خوان | خواندن پلاک خودرو در لحظه ورود |
| اپراتور ورودی | تأیید ورود عادی خودرو |
| اپراتور خروجی | تأیید خروج اضطراری خودرو |
| مدیر مالی | دریافت گزارش مالی از تراکنش‌ها |
| مدیر فنی | دریافت گزارش فنی از ترددها و جریمه‌ها |
| سامانه پلیس | استعلام وضعیت پلاک (مجاز/غیرمجاز/ممنوع) |
| سامانه پیامک | ارسال پیامک تأیید پرداخت به راننده |
| بانک | انجام پرداخت الکترونیکی هزینه‌ها |

## 📊 دیاگرام‌های ارائه‌شده

| نوع دیاگرام | توضیح |
|-------------|--------|
| DFD سطح صفر | نمای کلی سیستم و تعامل با تمام موجودیت‌های خارجی و مخازن داده |
| DFD سطح یک | تفکیک فرایندهای اصلی شامل: بررسی پلاک، استعلام پلیس، مدیریت ورود، ثبت تردد، پرداخت، و خروج اضطراری |
| نمودار Use Case | نمایش قابلیت‌های سیستم از دید بازیگران (راننده، اپراتورها، مدیران، سامانه‌های خارجی) |
| نمودار کلاس | ساختار شی‌گرای سیستم شامل کلاس‌های اصلی، ویژگی‌ها، متدها و ارتباطات بین آنها |
| نمودار ER | مدل داده‌ای شامل موجودیت‌ها (خودرو، راننده، تردد، پرداخت، جریمه، خودروی مجاز)، ویژگی‌ها و روابط یک‌به‌یک و یک‌به‌چند |
| نمودار توالی (۳ سناریو) | تعامل اشیا در سناریوهای: ورود خودروی مجاز، خروج عادی و پرداخت، خروج اضطراری |

## 🔄 سناریوهای اصلی سیستم

### ۱. ورود خودروی مجاز
1. راننده درخواست ورود می‌دهد.
2. دوربین پلاک خودرو را می‌خواند.
3. سیستم از سامانه پلیس استعلام می‌گیرد.
4. سیستم مجوز خودرو را در مخزن خودروهای مجاز بررسی می‌کند.
5. اپراتور ورودی ورود را تأیید می‌کند.
6. سیستم زمان ورود را در سوابق تردد ثبت می‌کند.
7. به راننده اجازه ورود داده می‌شود.

### ۲. خروج عادی و پرداخت
1. راننده درخواست خروج می‌دهد.
2. سیستم مدت توقف را محاسبه می‌کند.
3. هزینه از طریق بانک پرداخت می‌شود.
4. سیستم زمان خروج را ثبت می‌کند.
5. پیامک تأیید پرداخت برای راننده ارسال می‌شود.
6. به راننده اجازه خروج داده می‌شود.

### ۳. خروج اضطراری
1. اپراتور خروجی درخواست خروج اضطراری را تأیید می‌کند.
2. سیستم خروج اضطراری را در سوابق تردد ثبت می‌کند.
3. سیستم جریمه خروج اضطراری را ثبت می‌کند.
4. به راننده اطلاع داده می‌شود.
5. تأیید نهایی به اپراتور خروجی داده می‌شود.
